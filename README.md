A modern **full-stack developer portfolio website** showcasing projects, skills, and experience with a sleek **dark & golden UI**, smooth animations, and interactive features.

---

## 🚀 Features

* 🎯 Clean and modern UI design
* ⚡ Smooth scroll & reveal animations
* 🎨 Glitch text effects
* 📱 Fully responsive layout
* 📬 Contact form (Formspree integration)
* 🔗 Social media integration
* 💡 Optimized for performance & UX

---

## 🧰 Tech Stack

* **HTML** – Structure
* **CSS** – Styling & animations
* **JavaScript** – Interactivity
* **Font Awesome** – Icons
* **Formspree** – Contact form backend

---

## 🏗️ Architecture

👤 User → 🌐 Portfolio UI → 🎨 CSS Styling → ⚙️ JavaScript Interactions
│
├── ✨ Animations (Scroll + Glitch)
├── 📬 Contact Form (Formspree API)
└── 🔗 External Links (GitHub, LinkedIn)

---

## 🧠 How It Works

1. User visits the portfolio website
2. Smooth animations enhance UI experience
3. Projects and skills are displayed dynamically
4. User interacts with buttons and links
5. Contact form sends message via Formspree
6. Social links redirect to profiles

3. Select branch → `main`
